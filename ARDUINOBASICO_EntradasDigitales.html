Arduino Basico Entradas Digitales

04/10/2024

Resumen
En el concepto básico de entradas digitales con Arduino, se utilizan botones para controlar LEDs. Cada botón actúa como un interruptor que, al ser presionado, envía una señal de alto (HIGH) al microcontrolador, activando el LED conectado a un pin de salida digital. Cuando el botón no está presionado, la señal es baja (LOW) y el LED permanece apagado. Esta configuración permite entender cómo Arduino puede leer entradas digitales (0 o 1) y responder controlando dispositivos externos como LEDs, creando interacciones sencillas.
Introducción
En esta práctica, se utiliza un botón como entrada digital para controlar un LED, que actúa como una salida. El botón, al ser presionado, envía una señal de encendido (1) al microcontrolador, lo que activa el LED. Este tipo de configuración es común en sistemas de control básico, donde se requiere una interacción sencilla entre un usuario y un dispositivo mediante entradas y salidas digitales. El objetivo principal es aprender a utilizar pines digitales tanto para leer el estado de un botón como para controlar dispositivos externos, como un LED, de manera eficiente.

Materiales usados en esta practica:
  • Resistencias 220 Ohms 3
  • Arduino UNO 1
  • LEDs 3
  • Jumpers
  • Botones 2

Conclusiones
