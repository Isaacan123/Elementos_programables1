Resumen

para poder usar los servomotores y el potenciometro primero se necesita saber como es que funcionan y como van conectados para que no pueda hayar una falla o un corto circuito, asi que para poder iniciar la practica del servomotor tengo que iniciar analizando los cables y como van conectados y con el codigo que me dieron analizar a detalle y con diferentes funciones. En esta práctica, se utiliza un servomotor controlado por un potenciómetro conectado a un Arduino. El potenciómetro actúa como un divisor de voltaje, generando una señal analógica que Arduino lee y convierte en un valor digital. Este valor es utilizado para ajustar la posición del servomotor, permitiendo controlarlo de manera precisa. A medida que se gira el potenciómetro, el ángulo del servomotor cambia proporcionalmente, lo que demuestra cómo usar entradas analógicas para manipular dispositivos mecánicos, como servomotores.

Introducción
En esta práctica, se explora el control de un servomotor mediante un potenciómetro usando un Arduino. El potenciómetro, que sirve como entrada analógica, permite ajustar la posición del servomotor de manera continua. A medida que el potenciómetro se gira, varía la señal de voltaje que el Arduino lee, convirtiéndola en un valor digital que se utiliza para controlar el ángulo del servomotor. Esta práctica es útil para aprender cómo manejar entradas analógicas y generar movimientos precisos en dispositivos mecánicos, como los servos, abriendo la puerta a aplicaciones robóticas y de automatización.

Materiales usados en esta practica:
  • Arduino UNO 1
  • Servomotor 2
  • Jumpers
  • Potenciometro 2
